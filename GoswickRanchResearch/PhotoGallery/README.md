# Goswick Ranch — Photo Gallery

Photography library for the Goswick Ranch HOA website, Mayer, Arizona (86333).

All images are organized into five thematic subfolders. See [CATALOG.md](CATALOG.md) for a full annotated inventory with file sizes and subject descriptions.

## Folder Structure

| Folder | Images | Subject |
|---|---|---|
| `Aerial/` | 3 | Drone/aerial views of ridgelines, roads, and disturbed hillsides |
| `Creek/` | 1 | Big Bug Creek — rocks and cottonwood |
| `Landscape/` | 5 | Boulders, canyons, valleys, and Bradshaw Mountain ridges |
| `RanchLife/` | 2 | Cattle grazing, historic stone wall |
| `Seasons/` | 10 | Spring wildflowers, monsoon storms, fall color, winter snow |

**Total: 21 images — ~38 MB**

## Usage Notes

- All files are JPEG; suitable for web display after resizing/compression.
- Filenames are descriptive and kebab-cased — use them as alt-text references.
- The `Seasons/` folder is the largest and most varied; good source for hero/banner images.
- `Aerial/` images include a mining-disturbed hillside — use with appropriate context on the site.
