# Page 05: Board & Contact

*Priority: MVP | Audience: Residents and prospective buyers*

---

## Page Purpose

Provides contact information for the board, describes the board structure, and gives residents a way to reach the HOA without exposing email addresses to spam harvesters. Uses a Formspree contact form.

**H2:** Send Us a Message

| Field | Type | Notes |
|---|---|---|
| Full name | Text | Required |
| Email address | Email | Required |
| Property address / lot | Text | Optional — helps board route correctly |
| Subject | Select | General Question / Road Issue / Gate Issue / ARC Question / New Resident / Other |
| Message | Textarea | Required |

## Section 4: Road or Gate Issues

**H2:** Road and Gate Issues

> To report a road issue (washout, pothole, debris) or gate problem, use the contact form above and select "Road Issue" or "Gate Issue" from the subject menu. Please include the specific location and whether the issue poses an immediate safety concern.
>
> For emergency road conditions, call 911.
